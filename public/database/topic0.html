藏历是什么呢？

  • 天文历算探索小分队
  • 2025-04-15 09:27
  • 上海

藏历（藏文：ལོ་ཐོ་）是中国藏族的传统历法。作为一种阴阳合历，藏历兼具月亮的朔望周期（月相变化）和太阳的周年运动（季节变化），体现了丰富的天文学知识和藏族人民的智慧。

一、藏历的流派

（一）早期藏历

藏族历法历史悠久，据史料记载，公元前1000年以前，藏族已有自己的历法体系，最初是根据月亮圆缺变化来推算日、月、年。经过几个世纪的发展，藏族原始宗教——雍仲本教的信徒们逐渐掌握了精确推算冬至时间的方法，以冬至为一年的起点，并由此形成丰富的节日和仪轨。今天藏族的许多习俗与生活方式，都承袭自这一时期的古象雄文化。

公元7世纪吐蕃时期，唐朝的文成公主与金成公主先后入藏联姻，带来了内地的历法，推动了藏历的发展与完善。11世纪出版了著名的《敏竹林历书》，1206年则编定了更具系统性的《萨迦历书》。

（二）后期藏历（时轮历）

公元1027年起，源自印度《时轮经》的“时轮历”逐渐在藏区流行并沿用至今，已有一千多年的历史。藏历以“绕迥”纪年，即每60年为一个周期，类似内地的60甲子体系，以阴阳五行及十二生肖纪年。例如，2016年农历丙申年，在时轮历中称为“火猴年”。藏历与内地农历虽有相似性，但在月圆日（满月日）为每月十五日、新月日可能为初一或相差一天等细节上存在不同。此外，闰月的设置方式也与汉历有所区别，某些闰月年份可能会导致两个初一之间相差一个月或一个月加一天。现有历史记载表明，藏族历书最早出现在13世纪（元朝），到19世纪时已趋于完善。

随着时轮历法也随之在西藏各地盛行，高僧大德争相著书立说，形成了各种历算流派。如体系派和作用派、浦尔派和楚尔派、珠巴丹帝派、强达珠孜、松巴甘丹新算、智者心要、极显明、杰松达孜、殊胜七异、极同派、据鲁派等。出现如此众多流派的原因，其一是天文历算发展繁荣后各方学者出于对时轮根本经和摄略经的不同理解所致。其二是在实践当中为了与节气、天象、日月食等自然现象和观察结果相符合不断完善改进的结果，同时体现了学术上百花齐放、百家争鸣的研究状况。在各种历算流派中，以时轮摄略经为依据的极孜、浦尔派、楚尔派、甘丹新算影响最大，流传最广。

1. 极孜

极孜即作用派，最早由萨迦五祖之一扎巴坚赞尊者于1193年撰写了《集五基数、罗睺、五曜及日月食等要义之美卡加措》，后期各派学者编著的历算经典中都载有极孜的算法，而且计算方法和结果都完全一致，采取了积月闰余63、64时置闰的方法。

2. 浦尔派

主要依据浦巴伦珠加措于1447年编撰的《白莲亲教》计算，其后的历算经典名著《白琉璃》、《日光论》、《时轮历精要》等，都是依据此派算法程式，该派采用了积月闰余48、49时置闰的方法。现在各地通用的藏历，就是依据此派典籍计算。

3. 楚尔派

主要依据1732年噶玛俄列旦增编写的《满愿宝瓶》和1852年贡珠云旦加措编写的《嘉言集成》计算，此派算法的源头被认为是最早于1445年由楚布加央顿珠奥色编写的《历算大典》，该派采用了积月闰余65、66时置闰的方法。

4. 甘丹新算

依据松巴益西班觉编写的《黑白算要义集成明镜》计算，历元为1747年，采取了积月闰余46、47时置闰的方法。此派算法在现今的蒙古国依然在流行和使用。

藏历的发展历史

（一）阴阳合历的结构

藏历时轮历作为典型的阴阳合历，将月亮的朔望周期（月相变化）与太阳的周年运动（季节变化）统一起来，形成了高度复杂和科学的时间结构体系。

1. 月份与年份结构

藏历一年一般有12个太阴月，每月根据月相变化分为大月（30天）与小月（29天），全年约为354天。藏历以寅月为岁首，这与汉历的传统相同。为协调月与季节的错位，藏历周期性地插入闰月，以维持四季节律的稳定。

2. 闰日与差日规则

藏历还具有独特的“闰日”和“差日”制度来调整每日的序列。核心口诀是：“重者去大，缺者重小”。

  • 若日曜序重复，则去掉一个日期；
  • 若日曜序缺失，则补入一个日期；

这套规则保证了日序与太阴日之间的精确对应。

（二）干支纪年与岁次系统

自9世纪起，藏历吸收了汉历的干支纪年体系，称为“饶琼”，对应汉历的“六十花甲子”。其主要区别包括：

  • 天干以五行（木火土金水）表示，而非单纯甲乙丙丁；
  • 十二地支以生肖动物代表，如子鼠、丑牛等；
  • 例如，汉历“甲子年”在藏历中称为“木鼠年”。

（三）天文预测功能

藏历不仅用于纪年与节令，还具有极强的天文预测功能，包括：

  • 日月食的预测；
  • 五大行星（五星）位置的推算；
  • 二十八宿运行规律及节气漂移；
  • 指导藏医药采集时机与农牧活动。

（四）时间单位与计算体系

藏历采用细致入微的时间单位：

  • 1日 = 60刻（漏刻）
  • 1刻 = 60分（漏分）
  • 1分 = 6息（一呼一吸为一息）

虽然实际的太阴月平均长度约为29.53058日，但藏历统一设置为30太阴日，并以“差日”制度精确修正,，以确保历法与实际天象吻合。

三、藏历的地域特色与节气划分

由于藏地地域广阔，不同地区天文观测方法及气候条件各异，藏历中便纳入了各种丰富多样的谚语与地方特色，例如珞门法（观察禽鸟植物）、羌塘法（观察星象与风雪）、本象法（观察日月运行）、岗卓法（观察山湖牲畜）等。

在节气划分上，藏历不以简单的时间平均划分，而是依据“日宿”（太阳所在天球位置）确定节气，再与当地动植物变化结合。藏历还采用六季制，即春、后春、夏、秋、冬、后冬，更适合青藏高原复杂的气候状况。

四、藏历计算体系的原理与发展

藏历的计算体系融合了三大元素：藏族本土的物候历、从印度引进的时轮历、来自汉地的时宪历。

时轮历在计算时间上独特而严谨，规定每个太阴月为30太阴日，辅以闰日和差日制度。时轮历中有三种年、月、日（太阳、太阴、宫），并精确规定其换算关系：

  • 1太阳年 = 12太阳月 = 360太阳日
  • 1太阴年 = 12太阴月 = 360太阴日
  • 1宫年 = 12宫月 = 360宫日
  • 65宫日 = 67太阴日；64太阴日 = 63太阳日

此外，藏历中还设定了两个虚拟天体：罗睺与计都，用于精确预测日月食、交点等天文现象。

五、藏历的实际应用

藏历广泛用于授时、历法编制、藏医药诊治、气象预测及农牧生产。

  • 日常授时、历法编制与方位测定；
  • 藏医药诊治、择时与气象预测；
  • 通过鸟类迁徙和动植物观测，指导农牧生产和灾害预警；
  • 预测更长周期的气候变化和自然灾害（雪灾、旱涝、冰雹等）。
此外，藏历还融入五行、七曜、八卦、九宫、十二生肖、二十四节气等多种传统文化元素，体现了藏族天文历算文化的广博与深厚。

六、藏历的跨文化交流与发展

藏历历经历史演变，形成了藏族物候历、印度时轮历与汉族历法相融合的综合体系。特别是18世纪从汉地引入时宪历以来，更增强了藏历对日月食等天文现象预测的准确性与可靠性。由此可见，藏历不仅保留着本民族的智慧，也吸收了周边文化的精华，体现了多民族文化融合的特征。
