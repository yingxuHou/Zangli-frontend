时间单位与进制

时轮历中，从大到小，时间的基本单位如下：

  • 1年 = 12月
  • 1月 = 30日
  • 1日 = 60漏刻
  • 1漏刻 = 60漏分
  • 1漏分 = 6漏息
  • 息（呼吸顷）：最小时间单位，定义为健康壮年男子一呼一吸所需时间。

健康壮年男子一昼夜（24小时）平均呼吸21600次，即每分钟15息，符合现代医学每分钟14-18次的标准。

息以上的计时单位用滴漏系统。一昼夜为六十个流量，一个流量为六十两水，21600息=(60*60*6)息，滴漏一两水等于6息，这样就结合成为1-60-60-6的分法。为行文方便，不用流量或翻刻、一两水这些名称，而改用刻、分等通用的名称。但是须记住这里的1刻等于钟表上的24分，1分等于钟表上的24秒，1息等于钟表上的4秒。

在时轮历中“息”以下的单位没有单独的名称。其计算方法与汉历旧日的方法相似，展于代数系统，不是用小数点而是用分数来表示，其分母不是固定的，尽量选用能使分子为整数不再有零头的数值为分母，必要时用繁分数表示之。常用的分母有：13、67（可精确到0.0597秒）、707（可精确到0.0844毫秒）、18382 等。例如，体系派恒星年的长度为 365日16刻14分1/13，即365.2706451日。

空间单位与进制

在时轮历中，空间计量单位是以“星宿”为基本单位。在时轮历中，周天不是作为360°，而是均分为27等分，叫做“星宿”或“宿”。 星宿以下的单位与时间单位相似，有弧刻、弧分、弧息等，进制率为1:27:60:60:6。

  • 1周天 = 27星宿
  • 1星宿 = 60弧刻
  • 1弧刻 = 60弧分
  • 1弧分 = 6弧息

一周天27宿，每宿60弧刻，总计1620弧刻。

换算关系：

  • 1周天 = 1620弧刻
  • 12宫 = 1620弧刻
  • 1宫 = 135弧刻
  • 1星宿 = 60弧刻

近代世界通用的1°（360度制）= 藏历4.5弧刻。

特别注意：

  • "漏刻"是时间计量单位，"弧刻"是空间计量单位。
  • 二者在藏文中用同一个词，实际意义不同，需注意区分。

星宿与宫位划分

汉历中采用二十八宿系统：包括娄、胃、昴、毕、觜、参、井、鬼、柳、星、张、翼、轸、角、亢、氐、房、心、尾、箕、斗、牛、女、虚、危、室、壁、奎。而藏历多采用27宿体系，合并牛宿与女宿为一宿。

每宿分为60弧刻，进一步细分为四宿步，每宿步15弧刻。九宿步组成一宫。

曜、宿、刻、分、息的周期性

  • 曜（七曜）周期：7天
  • 宿（27宿/28宿）周期：27天左右
  • 刻与分：60进制
  • 息：6进制

息以上的计时单位遵循严格进制体系，而息以下的小单位（67、707分位等）则以精确推算为目的，采用变分母分数系统。
